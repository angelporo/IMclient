# Badge
> 徽章组件

## Usage
```javascript
<Badge
    {...options}
/>
```

## Options

Filed              | Type        | Description
:------------------|:------------|:----------------------
style              | React.Style | 文本样式
unReadMessageCount | Number      | 未读计数
height             | Number      | 可选：高度
anchorWidth        | Number      | 可选，固定长度，需要大于高度
autoMarginRight    | Number      | 可选，默认 false; 右侧自动偏移
