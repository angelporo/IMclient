# UI 基础组件
> 与业务无关，可单独发布成 `npm` 包，可复用

## Requirements
* react-native >= 0.41

## 组件列表
* [TabBar](./TabBar) - 底部 Tab 导航组件
* [Navigator](./Navigator) - 顶部导航视图
* [Badge]('./Badge') - 徽章组件
* [Button](./Button) - 按钮组
* [TextInput](./TextInput) - 输入框组
* [ListItem](./ListItem) - ListItem 样式组
* [Swipeout](./Swipeout) - 侧滑按钮组件

## 主题设置
* [Color](./Color) - 颜色定义文件
* [FontSize](./FontSize) -  字体定义文件
