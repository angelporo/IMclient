# TextInput
>输入框组件

## Usage
```javascript
<TextInput.Label
    labelText="用户名"
    placeholder="点击此处输入"
/>
```

## TextInput.Label
>带文字标签的输入框

Filed          | Type        | Description
:--------------|:------------|:-----------
style          | React.Style | 外框样式
labelText      | String      | 标签文本
labelStyle     | React.Style | 标签样式
textInputStyle | React.Style | 输入框样式

以及其他官方 `TextInput` 支持的属性
